# OLED SPI Driver for STM32L0

Driver para display OLED 64x48 usando SPI no STM32.

## Demonstração

![OLED Demo](images/oled_demo.jpg)

## Hardware usado

- STM32L011K4
- Display OLED 64x48 SPI
- NUCLEO board

## Features

- SPI interface
- Graphics primitives
- Text rendering
